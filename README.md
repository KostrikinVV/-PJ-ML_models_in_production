# -PJ-ML_models_in_production
Асинхронная система обработки сообщений, использующая RabbitMQ в качестве брокера сообщений. Этот учебный проект демонстрирует взаимодействие между HTTP-сервисом (FastAPI) и фоновыми обработчиками через очереди сообщений с поддержкой разделения по языкам (русский/английский).
